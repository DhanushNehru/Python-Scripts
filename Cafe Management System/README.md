# Cafe Management System

A simple **Cafe Management System** built with **Python Tkinter** that allows users to manage cafe items, calculate bills, and use an integrated calculator.  

This project is designed as a beginner-friendly GUI application to demonstrate billing operations and UI handling in Tkinter.

---

## Features
- **Cafe Items Panel**
  - Enter quantity for Tea, Coffee, Sandwich, Cake, Burger, Pizza, Fries, and Pepsi.
- **Billing System**
  - Calculates item cost, service charge, tax, subtotal, and final total bill.
- **Built-in Calculator**
  - Perform simple arithmetic operations directly inside the app.
- **Clear Function**
  - Reset all item entries and billing fields.
- **Exit Option**
  - Safe quit confirmation before closing the app.
- **GUI Design**
  - Clean, responsive interface with labels, entry fields, and buttons.

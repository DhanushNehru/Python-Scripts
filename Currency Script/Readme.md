# Currency Converter

A Python-based tool that allows users to convert currency from one country to another. The tool fetches real-time exchange rates via the Fixer.io or FloatRates API and offers offline support with caching functionality. This project includes two scripts: Currency.py for general conversion, and Currency_calculator.py for interactive rate fetching.

## Features

-Real-Time Currency Conversion: Convert between various currencies using live rates from the Fixer.io or FloatRates API.
-Offline Mode: Cached exchange rates are saved locally for offline usage or network failures.
-Error Handling: The program handles errors, such as invalid inputs, currency codes, and network issues, gracefully.
-Simple and Intuitive: Easy to use interface with minimal input needed from the user.

## Installation

Clone the repository to your local machine:

"git clone https://github.com/yourusername/currency-converter.git"

Install required Python libraries:

"pip install requests"

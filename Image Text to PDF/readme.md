## About:
This script adds image and text to a pdf file.

## Prerequisites
    Libraries: reportlab

#### Install :
```console
   pip install reportlab
   ```
 
## How to use:

* Replace <b>img.jpg</b> with the image you want to add.
* Add your text inside <b>input.txt</b>.
* Run in terminal 

```console
  python3 script.py
  ```
* Hurray!! Your PDF is generated.
